Exploring More ls Arguments
The argument -l, when used with the ls command, displays additional information with the file and directory names, such as file size. The argument -h makes the file size more easily readable for humans.

Listing Recursively and By Time
ls -r -t would organize the the directories in alphabetical order, and the files and directories within the directories would be sorted from most recently changed to the least recently changed.
